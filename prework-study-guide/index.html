HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content showen to the user.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • The style attribute can be used to affect a single h1 heading or can be implemented into the head element, embedding the entire stylesheet directly
  • The * selector declares all the elements of the page have the style properties margin and padding set at your choosing
  • Two selectors can combined into a single CSS rule with a comma, letting the number of selectors to be affected by the declarations
  • "text-align: center;": one way to center a title or heading on a webpage
  • The img selector assigns property values to any image element in the markup
  • Within the img selector, "display: block;" assigns each image to start on a new line
  • A link element creates the connection to an external file, to be placed in the head section where metadata is stored
  • The rel attribute specifies the relationship between the current and linked document
  • The href attribute specifies the location of the external source, uses a relative path, i.e. derived from the current location
  • The .card selector assigns any HTML elemment the property assignments given from the CSS

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit: commits changes
  • git commit -m "phrase": commits changes with a message

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The const keyword creates a variable that can be assigned.
  • The console.log method outputs a message to the web console by adding an argument.
  • The script.js file is linked within the body element.
  • Conditional statments and loops control the order of code execution.
  • The if statment is a way of introducing decision-making, the code will only execute if the statment meets a condition or is true.
  • The else if statment can be use to have more than one condition.
  • An array is a single variable used to hold a group of data, this data is usually related in some way.
  • The index is the location of the data in the array and is identified with a unique number.
  • A for loop performs a task on all the items in an array by allowing a single block of code be executed over and over.
  • A for loop consists of three parts; a starting point (let x=0), a condition, and an iteration.
  • A function is a set of instructions that tells the computer how to perform a certain task.
